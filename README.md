# API REST – Gestión de Tareas

API REST desarrollada en Python utilizando Flask para la gestión de tareas.

## Tecnologías
- Python
- Flask
- JSON
- Git

## Funcionalidades
- Listar tareas
- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas
